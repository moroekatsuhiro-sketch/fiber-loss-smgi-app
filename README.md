# SM/GI 光ケーブル規格値計算 fix11-reset-state-verified

## fix11 の目的

fix10 までの「消えない対策」に対して、明示的な「入力リセット」で前回測定値が作業画面へ残らないことを最優先で再設計した版です。

## 主な仕様

- ケーブル長 m 直接入力を最優先。未入力時のみ始端/遠端レングスマーク差を使用。
- 測定値・校正値・規格値は小数第3位以下切り捨て、小数第2位表示。
- 入力値全体を自動保存。測定値だけではなく、基本情報・波長別設定・計算状態も保存。
- 起動時に通常ドラフトがあれば自動復元。確認ダイアログは出さない。
- 軽量バックアップは最大2世代、15秒間隔、変更時のみ。
- バックアップは手動復元のみ。正式履歴には自動保存しない。
- 入力リセット/新規入力では、直前データをバックアップへ退避してから、作業中状態を全層で明示的に消去。
- リセット後は通常ドラフトを削除し、waveDraft / latestCalculation / DOM入力欄 / 入力結果一覧を初期化。
- PWA更新確認は3分ごと。更新適用はユーザー操作時のみ。
