# 光ケーブル規格値計算 PWA 試作版

## 内容

スマホ向けの光ケーブル規格値計算WEBアプリです。

主な機能：

- SM/GI通常計算
- SI専用計算
- CF系(圧着)の1km超過エラー
- 芯線数に応じた測定値入力
- 芯線ごとのOK/NG判定
- localStorage履歴保存
- 履歴削除
- CSV出力
- JSONバックアップ/読込
- PWA土台

## ファイル

- `index.html`
- `style.css`
- `app.js`
- `manifest.json`
- `service-worker.js`
- `icon-192.png`
- `icon-512.png`

## 実スマホ確認

PCでこのフォルダを開き、以下を実行します。

```bash
py -m http.server 8000
```

または、

```bash
python -m http.server 8000
```

PCのIPアドレスを調べ、スマホを同じWi-Fiに接続して以下のようにアクセスします。

```text
http://PCのIPv4アドレス:8000
```

例：

```text
http://192.168.1.23:8000
```

## 注意

- カメラOCRは未実装です。始端盤名・遠端盤名横の「写真」ボタンは次段階用の仮ボタンです。
- PWAのホーム画面追加やカメラ機能を本格確認する場合は、HTTPS環境が必要です。
- 履歴はブラウザ内に保存されます。ブラウザデータ削除や端末変更で消える可能性があります。

## 追加仕様：区間名の自動生成

- 始端盤名と遠端盤名を入力すると、区間名を自動で「始端盤名 ～ 遠端盤名」の形式にします。
- 例：始端盤名「1F MDF」、遠端盤名「3F IDF」→ 区間名「1F MDF ～ 3F IDF」
- 区間名を手入力で変更した場合は、その手入力値を優先し、自動上書きしません。

## 追加仕様：盤名OCR

- 始端盤名・遠端盤名の「写真」ボタンから写真を選択/撮影できます。
- Tesseract.js を使って、ブラウザ上で OCR を実行します。
- OCR結果は候補表示し、作業者が選択または修正して盤名欄へ反映します。
- 完全自動保存ではなく、誤読防止のため確認式です。
- 初回読み込み時は OCR ライブラリの読み込みに時間がかかります。
- 現在は CDN 版 Tesseract.js を読み込むため、OCR利用時はインターネット接続が必要です。
- カメラ直呼び出しではなく、`input type="file" accept="image/*" capture="environment"` を使った写真選択/撮影方式です。

## 追加仕様：自由線番・双方向測定・測定メモ

- 芯線ごとの行に「始端側線番」と「遠端側線番（相手方）」を追加しました。
- 初期値は 1, 2, 3... ですが、自由に変更できます。
- 測定値は「始端 → 遠端」と「遠端 → 始端」の双方向で記録できます。
- 各方向の測定値ごとにメモ欄を追加しました。
- CSV出力にも、始端側線番、遠端側線番、双方向測定値、双方向判定、各メモを出力します。

## 追加仕様：線番設定・片方向/双方向・1行1メモ

- 測定方向を「双方向」「始端 → 遠端」「遠端 → 始端」から選択できます。
- メモは測定値ごとではなく、線番1行につき1個に変更しました。
- 始端側芯数と遠端側芯数を別々に設定できます。
- 基本は始端側芯数と遠端側芯数は同じ値で使います。
- 始端側開始線番、遠端側開始線番を設定できます。
- 開始線番に「4」を入れると、以降は 4, 5, 6... の連番で自動生成します。
- 「A-01」のように末尾が数字の場合は、A-01, A-02, A-03... のように自動生成します。
- 相手方の芯数が違う場合は、始端側芯数・遠端側芯数を個別に変更して記録できます。
- CSV出力にも、測定方向、始端側芯数、遠端側芯数、始端側線番、遠端側線番、双方向測定値、線番メモを出力します。


## オフラインアプリ化版

この版は、PWAとしてスマホのホーム画面に追加して使う想定です。

### 方針

- 完全オフライン運用を優先
- OCRは未搭載
- 盤名は手入力
- 計算、線番設定、片方向/双方向測定、履歴保存、CSV出力、JSONバックアップはオフライン対応

### スマホへ入れる手順

1. PCで VS Code + Live Server などを使って `index.html` を開く
2. スマホで同じURLを開く
3. ブラウザメニューから「ホーム画面に追加」または「アプリをインストール」を選択
4. 一度アプリを開いて読み込み完了させる
5. 以後はホーム画面アイコンから起動

### 注意

- 初回読み込み時だけ、PCのLive Serverや配信URLにアクセスできる必要があります。
- 一度読み込んでService Workerが登録されると、アプリ本体はオフラインでも起動できます。
- 履歴は端末ブラウザ内に保存されます。定期的にJSONバックアップまたはCSV出力してください。
- ブラウザのサイトデータ削除、キャッシュ削除、端末変更で履歴が消える可能性があります。


## SM/GI専用版

この版は、SI系を分離した SM/GI 専用のオフラインPWAです。

### 対象

- SM 1310nm
- SM 1550nm
- GI 850nm
- GI 1300nm

### 分離理由

SI系は測定表記・芯線の数え方・計算方式がSM/GIと異なるため、現場入力ミスを避ける目的で別アプリに分けます。

### この版に含めないもの

- SI規格 S01-L2
- SI規格 DL-72
- SI規格 DLC-L2
- CF系(圧着)
- SI用の測定表記
- SI用の芯線数え方

SI系は別途、SI専用アプリとして仕様確定後に作成します。

# SM/GI 光ケーブル規格値計算 オフラインPWA 最終版

## この版の方針

- SM/GI専用です。
- SI系は別アプリとして分離します。
- 計算、測定入力、履歴保存、履歴編集、CSV出力、JSONバックアップはオフライン運用前提です。
- OCRはオンライン時のみ使える補助機能です。オフライン現場では手入力してください。

## 主な機能

- SM 1310nm / 1550nm
- GI 850nm / 1300nm
- 両波長同時計算
- 波長ごとの始端側・遠端側測定入力
- 始端側芯数 / 遠端側芯数の個別設定
- 開始線番から自動連番
- 芯線数変更時の測定値保持
- 入力結果一覧
- NG赤表示
- 履歴保存
- 履歴編集
- 履歴1件ごとのCSV出力
- 全履歴CSV出力
- JSONバックアップ / 読み込み
- PWAホーム画面追加

## スマホへ入れる手順

1. PCでこのフォルダを VS Code で開く
2. Live Server で `index.html` を開く
3. スマホでPCのURLへアクセスする
4. Chromeのメニューから「ホーム画面に追加」または「アプリをインストール」を選ぶ
5. 一度起動して画面を読み込ませる
6. Wi-Fiやモバイル通信を切って、ホーム画面アイコンから起動確認する

## 注意

- ブラウザのサイトデータ削除をすると、端末内の履歴が消える可能性があります。
- 定期的にJSONバックアップまたはCSV出力してください。
- PWA更新時は、古いService Workerキャッシュが残ることがあります。その場合はサイトデータ削除か、別名フォルダ/別URLで再読み込みしてください。



# SM/GI オフラインOCR試験版

## この版の目的

- SO-52Fなどの現場スマホで、完全オフラインOCRを試験するための版です。
- 計算・履歴・CSV・JSONバックアップは従来どおりオフライン動作します。
- OCRは `vendor/tesseract/` に必要ファイルを配置するとオフライン動作します。

## OCR対象

- 現場名
- 始端盤名
- 遠端盤名

## OCRファイル配置場所

`vendor/tesseract/` に以下を配置してください。

- `tesseract.min.js`
- `worker.min.js`
- `tesseract-core.wasm.js`
- `tesseract-core.wasm`
- `jpn.traineddata.gz`
- `eng.traineddata.gz`

## アプリ側のOCR参照先

- `tesseract.min.js`：`./vendor/tesseract/tesseract.min.js`
- `workerPath`：`./vendor/tesseract/worker.min.js`
- `corePath`：`./vendor/tesseract/tesseract-core.wasm.js`
- `langPath`：`./vendor/tesseract/`

## 注意

このZIPにはOCR本体ファイルは同梱していません。
この実行環境から外部のOCRデータを直接取得できないためです。

OCRファイルを配置した後、再度ZIP化してスマホへ入れるか、VS Code + Live Serverで読み込んでホーム画面へ追加してください。



## OCRファイル取得スクリプト

この版には `download-ocr-files.ps1` を追加しています。

Windowsでこのファイルを PowerShell 実行すると、`vendor/tesseract/` にオフラインOCR用ファイルを取得します。

取得完了後、VS Code + Live Serverで `index.html` を開き、スマホ側でホーム画面に追加してください。
