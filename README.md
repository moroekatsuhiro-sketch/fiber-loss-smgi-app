# SM/GI 光ケーブル規格値計算 fix9-auto-restore-light-backup

## 反映内容

- fix8をベースに、起動時確認を減らすため「前回ドラフトの自動復元」に変更
- 入力値全体の通常ドラフト保存を継続
- 軽量バックアップを追加
  - 最大2世代
  - 15秒間隔
  - 変更があった時のみ保存
  - pagehide / beforeunload / 画面非表示 / PWA更新適用前は即時退避
- バックアップ復元は手動
- 「新規入力（現在を退避）」ボタンを追加
  - 現在入力をバックアップに退避してから新規入力状態にする
- 正式履歴には自動保存しない
  - 正式履歴は従来通り「履歴に保存」押下時のみ
- 3分ごとのPWA更新確認を継続
- 更新適用前に入力値全体を通常ドラフト＋バックアップへ保存
- バージョン表示を継続
- Service WorkerのSKIP_WAITING処理を明示実装

## 保存対象

ドラフト保存・バックアップ保存・更新前保存・離脱前保存では、測定値だけでなく以下を保存対象にする。

- 工事番号
- 現場名
- 区間名
- 区間名が手入力かどうか
- 始端盤名
- 遠端盤名
- ケーブル長 m
- 始端レングスマーク
- 遠端レングスマーク
- ケーブル種類
- 波長
- 融着点数
- コネクタ数
- 区間メモ
- 波長別 始点校正値
- 波長別 終点校正値
- 波長別 芯線数
- 波長別 開始線番
- 各芯線の測定値
- 各芯線メモ
- 最新の規格値計算状態
- 編集中履歴ID

## ケーブル長の優先順位

1. ケーブル長 m が入力されている場合は、その値を最優先で使用
2. ケーブル長 m が空欄の場合のみ、始端LM・遠端LMの差を使用
3. どちらも使えない場合はエラー

内部計算では m を km に変換して規格値計算に使用する。

## 計算式

規格値 = ケーブル長(km) × ケーブル損失 + 融着点数 × 0.15 + コネクタ数 × 0.35

## マスター

SM 1310nm:
- ケーブル損失 0.50 dB/km
- 融着損失 0.15 dB/点
- コネクタ損失 0.35 dB/個

SM 1550nm:
- ケーブル損失 0.30 dB/km
- 融着損失 0.15 dB/点
- コネクタ損失 0.35 dB/個

GI 850nm:
- ケーブル損失 3.00 dB/km
- 融着損失 0.15 dB/点
- コネクタ損失 0.35 dB/個

GI 1300nm:
- ケーブル損失 1.00 dB/km
- 融着損失 0.15 dB/点
- コネクタ損失 0.35 dB/個

## 小数処理

- 規格値・測定値・校正値は小数第3位以下を切り捨て、小数第2位表示
- OK/NG判定は切り捨て後の測定値と切り捨て後の規格値で比較
- 浮動小数点誤差で 1.15 → 1.14 のように落ちないよう補正

## 注意

バックアップは同一ブラウザのlocalStorage内に保存するため、サイトデータ削除・PWA削除・端末故障・ブラウザストレージ破損には対応できない。
正式な保全が必要な場合は、履歴保存・JSONバックアップ・CSV出力を併用する。
