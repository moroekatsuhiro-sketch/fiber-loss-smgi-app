# SM/GI 光ケーブル規格値計算 fix5

## 試作内容

- 校正値を波長ごとに入力
- 波長を単独選択した場合は、その波長分だけ校正値・芯線数・測定値一覧を表示
- 両波長選択時は、両方の波長分を表示
- 測定値一覧は1〜4のような連番一覧表示
- 始端側→遠端側の一覧には始点校正値だけ表示
- 遠端側→始端側の一覧には終点校正値だけ表示
- 校正値は小数第2位表示
- 測定値・校正値は小数第3位以下を切り捨て、小数第2位表示
- 校正値は記録用のみ。規格値計算・OK/NG判定には使わない
- 融着点数の初期値は2
- 履歴保存前でも入力結果一覧を表示
- CSVでは測定方向ごとに必要な校正値だけ出力

## 計算式

規格値 = ケーブル長(km) × ケーブル損失 + 融着点数 × 0.15 + コネクタ数 × 0.35

## マスター

SM 1310nm:
- ケーブル損失 0.50 dB/km
- 融着損失 0.15 dB/点
- コネクタ損失 0.35 dB/個

SM 1550nm:
- ケーブル損失 0.30 dB/km
- 融着損失 0.15 dB/点
- コネクタ損失 0.35 dB/個

GI 850nm:
- ケーブル損失 3.00 dB/km
- 融着損失 0.15 dB/点
- コネクタ損失 0.35 dB/個

GI 1300nm:
- ケーブル損失 1.00 dB/km
- 融着損失 0.15 dB/点
- コネクタ損失 0.35 dB/個

- 復元時に `collectWaveInputs()` が空の画面状態で `waveDraft` を上書きしないよう修正

  3. 波長別設定欄を描画
  4. 測定値入力欄を描画
  5. 復元モード解除
  6. 入力結果一覧を描画
- 測定値入力欄と入力結果一覧の両方に復元値が残るよう修正
